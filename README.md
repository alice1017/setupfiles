# setupscripts

the **setupscripts** contains the shell scripts for **setup to the development
environ**.

## Features

### install following application

* git
* docker-engine
* zsh
* vim
* anyenv

### download files

* dotfiles - clone from [alice1017/dotfiles](https://github.com/alice1017/dotfiles)

## Script stracture

```
setup-zsh.sh
    |-- setup-zsh_linux.sh
    |-- setup-zsh_mac.sh
```
